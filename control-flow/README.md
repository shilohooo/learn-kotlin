# Kotlin control flow

## 创建值范围

在 Kotlin 中, 创建值范围最常见的办法是使用 .. 操作符.

|           语法            |          范围          |     备注      |
|:-----------------------:|:--------------------:|:-----------:|
|         `1..4`          |     `1, 2, 3, 4`     |             |
|         `1..<4`         |      `1, 2, 3`       |             |
|      `4 downTo 1`       |     `4, 3, 2, 1`     |     倒序      |
|      `1..5 step 2`      |      `1, 3, 5`       |    步长为2     |
|       `'a'..'d'`        | `'a', 'b', 'c', 'd'` | Char 类型的值范围 |
| `'z' downTo 's' step 2` | `'z', 'x', 'v', 't'` |             |
