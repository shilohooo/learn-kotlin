# Kotlin Collection

Kotlin 提供了以下集合类型来组织数据元素

|  集合类型  |                     描述                     |
|:------:|:------------------------------------------:|
| `List` |                 有序的元素组成的集合                 | 
| `Set`  |               唯一的，无序的元素组成的集合               | 
| `Map`  | 一组键值对（key-value pair），其中键是唯一的，并且每个键对应到唯一的值 | 

每个集合类型都是可变的，或只读的
